Theme Test Page

Light/Dark Theme Support: This page demonstrates Pinora's automatic theme switching based on your system preferences.

To test:

  • On macOS: Go to System Preferences → General → Appearance and switch between Light/Dark
  • On Windows: Go to Settings → Personalization → Colors and change the theme
  • On Linux: Change your system theme settings
  • Browser override: In Chrome DevTools (F12) → click the three dots → More tools → Rendering → Emulate CSS media feature "prefers-color-scheme"

The page should automatically switch between light (white background) and dark (dark background) themes.
